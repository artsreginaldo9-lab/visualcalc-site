Dúvidas

Perguntas frequentes

Preciso instalar alguma coisa? +
Não. O VisualCalc ERP é 100% online — funciona direto no navegador, sem instalar nada. Acesse de qualquer computador, tablet ou celular, de qualquer lugar.
Funciona para ACM, fachadas e comunicação visual? +
Sim, foi feito exatamente para isso. O sistema calcula automaticamente orçamentos de ACM, metalon, lona, banner, adesivo vinil, placa PVC, letreiros e outros materiais. Você cadastra seus materiais com custo por m² e o sistema faz o resto.
Tem fidelidade ou contrato mínimo? +
Não. O VisualCalc é mensalidade sem fidelidade. Cancele quando quiser, sem burocracia. Seus dados ficam disponíveis por 30 dias após o cancelamento para que você possa exportar tudo.
Como funciona o Kanban de produção? +
Quando um orçamento é aprovado, o pedido entra automaticamente na fila do Kanban. São 7 etapas: Fila → Impressão → Acabamento → Montagem → Pronto → Entrega → Entregue. O sistema calcula o prazo, emite alertas de atraso e registra cada etapa com data e responsável.
Como funciona a emissão de NF-e? +
Disponível no plano Avançado. O sistema integra com a Nuvem Fiscal (parceiro autorizado pela SEFAZ). Você configura seu certificado A1 e CNPJ uma vez. Depois, emite NF-e direto de qualquer pedido aprovado, com DANFE em PDF gerado automaticamente.
Meus dados ficam seguros? +
Sim. Cada empresa tem seus dados completamente isolados com RLS (Row Level Security) no banco de dados. Uma empresa nunca consegue ver os dados de outra. Toda ação no sistema é registrada no Event Log para auditoria completa.
Tem suporte? Como funciona? +
Sim. Suporte via WhatsApp diretamente com a equipe que desenvolveu o sistema. Sem robô, sem fila. Respondemos em minutos. Atendimento de segunda a sábado, das 8h às 18h.
O sistema funciona para serralherias? +
Sim. O VisualCalc atende gráficas, comunicação visual, ACM, fachadas, adesivos, impressão digital, plotagem e serralherias. Qualquer negócio que trabalha com orçamentos por m², dimensões ou produtos personalizados encontra valor no sistema.
Posso ter múltiplos usuários? +
Sim. O plano Básico tem 1 usuário, o Profissional tem 3 usuários e o Avançado tem usuários ilimitados. Cada usuário tem acesso com login próprio e você pode definir permissões por perfil.
